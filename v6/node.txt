Ignore node modules after adding

1. Rename node_modules to something else e.g backup_node_modules
2. Run git rm -r node_modules 
3. Add node_modules to .gitignore
4. Run git add .gitignore
5. Commit 
6. Rename node_modules back e.g backup_node_modules to node_modules
